Modular Research
Doc Writer

Biến các yêu cầu nghiên cứu phức tạp thành hệ thống Markdown module hóa. Tối ưu hóa tuyệt đối cho AI Agent và kho tri thức hiện đại.

Bắt đầu ngay Tìm hiểu thêm

4 Lớp chuẩn hóa

01. Skill

Định nghĩa hành vi tái sử dụng cho Agent. Hướng dẫn Agent cách tư duy chuẩn MRM.

02. Workflow

Quy trình từ Scoping đến Assemble báo cáo một cách khoa học và nguyên tử.

03. Contracts

Thiết lập chuẩn Output (Frontmatter, TL;DR) và bộ Rubric đánh giá tự động.

04. Adapters

Tích hợp trực tiếp vào Claude Code, Gemini, GitHub Copilot và Cursor IDE.

Kiểm soát bằng CLI

Đảm bảo tính nhất quán của dữ liệu nghiên cứu với bộ validator mạnh mẽ.

$ python mrm_validator.py validate research/
[PASS] 01-intro.md (Modular Frontmatter)
[PASS] 02-methodology.md (Context Safe)
[WARN] Missing cross-reference in 03-results.md
$ python mrm_validator.py assemble research/ final_report.md
